## Run the app
To spin up the development server, follow these steps:

1. Install the dependencies:
  ```bash
  npm install
  ```
2. Start the development server:
  ```bash
  npm run dev
  ```
3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
